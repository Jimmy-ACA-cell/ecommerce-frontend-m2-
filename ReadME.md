# Specialty Coffee Store — Frontend (MVP)

Proyecto: e-commerce frontend (Módulo 2) — **Sello de Autor, Specialty Coffee Store**  
Tema: Importación y venta de máquinas espresso, molinos, tostadoras, accesorios y café de especialidad.

## Funcionalidades

- Grid de productos en Home (renderizado desde `js/index.js`).
- Página de detalle (productos.html?id=...).
- Carrito con: agregar, aumentar, disminuir, eliminar, vaciar y total calculado.
- Contador del carrito en la navbar (badge).
- Responsive (mobile-first) usando Bootstrap.

## Requisitos cumplidos (MVP)

- HTML5 semántico (header, nav, main, section, article, footer).
- Bootstrap por CDN (CSS + JS).
- JS básico: `querySelector`, eventos `click`, arrays/objetos, `localStorage`.
- Carrito funcional con total calculado.
- Responsive: pruebas en móvil y escritorio, OK.
